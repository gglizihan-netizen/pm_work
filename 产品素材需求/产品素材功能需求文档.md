# 产品素材功能需求文档

## 一、需求概述

### 1.1 需求背景

目前AI编标产品主要服务于工程类项目，其核心能力是基于施工组织设计方案自动生成技术标。然而，**货物类项目**（如办公设备采购、家具采购、医疗器械采购等）与工程类项目存在本质差异：

- 工程类项目强调施工方案、工艺流程、进度计划
- **货物类项目强调产品本身的参数响应、技术规格符合性、供货能力证明**

在货物类项目中，投标人需要根据招标文件对产品的具体要求，结合自身所投产品的真实信息进行编写。如果AI生成内容时缺乏实际产品信息作为依据，容易产生"参数不符合"、"产品与需求不匹配"等问题。

### 1.2 需求价值

- **提升货物标编写准确性**：让AI基于真实产品参数生成内容，避免空泛描述
- **提高投标效率**：产品信息一次录入多次复用，减少重复整理产品资料的时间
- **降低废标风险**：确保技术响应与产品实际参数一致，减少因参数不符导致的废标

### 1.3 需求目标

构建一套产品素材管理体系，支持用户：
1. 便捷地录入和维护产品信息（支持手动录入和AI智能提取）
2. 在创建项目时灵活挑选所需产品
3. 在正文生成时自动引用产品信息，生成符合实际的技术响应内容

## 二、产品信息结构

### 2.1 字段定义

| 字段名称 | 类型 | 必填 | 说明 |
|---------|------|------|------|
| 产品名称 | 文本 | 是 | 产品的标准名称 |
| 产品类型 | 文本 | 是 | 用于分类，如"办公家具"、"电子设备"等 |
| 型号规格 | 文本 | 是 | 产品的型号、规格信息 |
| 单位 | 文本 | 是 | 如"台"、"套"、"件"等 |
| 产品品牌 | 文本 | 否 | 品牌名称 |
| 技术参数 | 文本 | 否 | 纯文本形式，描述产品技术规格 |
| 详细介绍 | 文本 | 否 | 产品的功能、特点、应用场景等描述 |
| 产品图片 | 图片 | 否 | 与私有图库打通，可多选 |
| 产品彩页 | 图片 | 否 | 与私有图库打通，可多选 |
| 认证资质 | 文本 | 否 | 如3C认证、ISO认证等 |

### 2.2 数据范围

- **归属**：跟随企业/团队，非个人

## 三、录入方式

### 3.1 单条录入

**入口**：产品素材管理页面，点击「新增产品」按钮

**交互流程**：

1. **打开录入弹窗**
   - 点击「新增产品」按钮，弹出录入弹窗（模态框）
   - 弹窗标题：新增产品
   - 弹窗尺寸：建议宽度600-800px，高度自适应，内容过多时内部滚动

2. **表单布局**
   - 采用两列布局，提升屏效
   - 左列：产品名称、产品类型、型号规格、单位、产品品牌
   - 右列：技术参数、详细介绍、认证资质
   - 底部横跨两列：产品图片、产品彩页

3. **字段录入说明**

   | 字段 | 录入方式 | 特殊说明 |
   |------|---------|---------|
   | 产品名称 | 单行文本输入 | 限制长度100字符 |
   | 产品类型 | 下拉选择 + 支持手动输入 | 下拉展示历史用过的类型，方便归类 |
   | 型号规格 | 单行文本输入 | 限制长度200字符 |
   | 单位 | 下拉选择 + 支持手动输入 | 预设常用单位：台、套、件、个、米、平方米等 |
   | 产品品牌 | 单行文本输入 | 限制长度50字符 |
   | 技术参数 | 多行文本输入 | 5行高度，支持换行，限制2000字符 |
   | 详细介绍 | 多行文本输入 | 8行高度，支持换行，限制5000字符 |
   | 认证资质 | 多行文本输入 | 3行高度，支持换行，限制1000字符 |
   | 产品图片 | 图片选择器 | 点击打开图库选择弹窗，支持多选，最多5张 |
   | 产品彩页 | 图片选择器 | 点击打开图库选择弹窗，支持多选，最多5张 |

4. **图片选择交互**
   - 点击图片选择区域，弹出图库选择弹窗
   - 弹窗内展示私有图库中的所有图片
   - 支持图片预览、多选（checkbox）、搜索
   - 已选图片在选择区域以缩略图形式展示
   - 每张缩略图右上角显示删除按钮，点击移除关联

5. **表单操作按钮**
   - 底部固定操作栏
   - 左侧：「取消」按钮，点击关闭弹窗，不保存
   - 右侧：「保存」按钮（主按钮），点击校验必填项，通过后保存并关闭弹窗

6. **校验规则**
   - 必填项为空时，点击保存提示"请填写XXX"
   - 字符超限提示"XXX不能超过XXX字"
   - 保存成功后在产品列表中展示新产品

### 3.2 AI分析录入

**支持格式**：PDF、Word（doc/docx）、Excel（xls/xlsx）

**处理流程**：
1. 用户上传文件（支持批量上传多个文件）
2. 系统将文件内容转为文本
3. AI分析文本，按产品信息结构提取数据
4. 提取结果直接入库，在产品列表中展示
5. 用户可在列表中对提取结果进行编辑或删除

**提取失败处理**：
- 若AI无法提取结构化信息，提示用户"未能识别有效产品信息，建议手动录入"

## 四、产品信息维护

### 4.1 列表展示

- 以表格形式展示，支持分页
- 表头字段：产品名称、产品类型、型号规格、单位、产品品牌
- 支持按产品名称、产品类型、型号规格搜索

### 4.2 基础操作

- **新增**：跳转至单条录入表单
- **编辑**：弹出编辑弹窗，可修改所有字段
- **删除**：单条删除 + 批量删除
- **查看详情**：点击产品名称查看完整信息

### 4.3 批量操作

- 列表支持多选（checkbox）
- 批量删除功能

---

## 五、智能引用（项目级挑选）

### 5.1 挑选入口

**位置**：创建项目页面的「素材库挑选」区域

**交互设计**：
- 点击「素材库挑选」弹出挑选弹窗
- 弹窗顶部增加切换标签：「文档素材 | 产品素材」
- 默认展示文档素材标签

### 5.2 产品素材挑选页

**表头字段**：
- 复选框（全选/单选）
- 序号
- 产品名称
- 产品类型
- 型号规格
- 产品品牌

**搜索功能**：支持按产品名称、产品类型、型号规格筛选

### 5.3 已选素材区域

**位置**：创建项目页面，「素材库挑选」按钮下方

**展示效果示意**：

**具体描述**：

- **容器样式**：
  - 圆角边框卡片，浅色背景（如#f5f5f5）
  - 顶部标题"已选素材（N）"，N为已选总数（文档+产品）

- **素材条目样式**：
  - 每条素材占一行，垂直排列
  - 文档素材和产品素材样式统一，通过标签区分类型
  - 类型标签：【文档】蓝色，【产品】绿色
  - 条目之间无分隔线，通过上下间距区分

- **产品素材条目内容**：
  - 第一行：【产品】产品名称 ｜ 产品类型 ｜ 型号：XXX
  - 第二行（缩进对齐）：品牌：XXX
  - 使用"｜"作为分隔符，信息紧凑展示

- **移除操作**：
  - 每条素材右侧显示"×"删除图标
  - 点击移除该条素材（不弹确认，直接移除）

- **空状态**：
  - 未选择任何素材时，区域隐藏或显示"暂无已选素材"

**选择限制**：
- 单项目最多选择50款产品
- 超出限制时提示"单个项目最多选择50款产品"

## 六、Prompt注入策略

### 6.1 注入位置

产品素材信息与文档素材信息统一放在指令末尾。

### 6.2 组织格式

```
【参考资料】

一、企业素材库文档
[文档素材内容]

二、产品素材信息
本项目涉及以下产品：

【产品类型A】
- 产品名称：XXX
  型号规格：XXX
  单位：XXX
  品牌：XXX
  技术参数：XXX
  详细介绍：XXX
  认证资质：XXX

【产品类型B】
- 产品名称：YYY
  型号规格：YYY
  ...

  请根据上述产品信息编写正文，涉及具体产品时请准确引用参数。
```

### 6.3 注入范围

- **MVP阶段**：全量注入已选的所有产品信息
- 由模型自主决策在哪些章节引用哪些产品

---

## 七、与私有图库的打通

### 7.1 产品图片/彩页入库

- 在录入产品时选择图片，图片同时进入私有图库
- 图片名称自动带入产品名称作为前缀（如"办公椅_主图"）
- 图片描述默认为"产品图片"或"产品彩页"

### 7.2 图片管理

- 在产品详情中可查看、增删关联的图片
- 删除产品素材时，不级联删除图库中的图片（避免影响其他引用）

---

## 八、异常处理

| 场景 | 处理方式 |
|------|---------|
| AI提取无结果 | 提示"未能识别有效产品信息，建议手动录入" |
| 单项目选择超50款 | 提示"单个项目最多选择50款产品" |
| 文件上传失败 | 提示具体失败原因，支持重试 |
