## 3. 需求概述

### 3.1 需求背景

AI编标产品目前主要服务于工程类项目，核心能力是基于施工组织设计方案自动生成技术标。然而，**货物类项目**（如办公设备采购、家具采购、医疗器械采购等）与工程类项目存在本质差异：

| 项目类型 | 核心要求 | 当前问题 |
|----------|----------|----------|
| 工程类项目 | 施工方案、工艺流程、进度计划 | 已有成熟方案 |
| 货物类项目 | 产品参数响应、技术规格符合性、供货能力证明 | 缺乏产品信息支撑 |

在货物类项目中，投标人需要根据招标文件对产品的具体要求，结合自身所投产品的真实信息进行编写。当前AI生成内容缺乏实际产品信息作为依据，容易产生"参数不符合"、"产品与需求不匹配"等问题。

### 3.2 需求价值

| 价值维度 | 具体收益 |
|----------|----------|
| 提升准确性 | AI基于真实产品参数生成内容，避免空泛描述 |
| 提高效率 | 产品信息一次录入多次复用，减少重复整理时间 |
| 降低风险 | 确保技术响应与产品实际参数一致，减少废标风险 |

### 3.3 核心目标

1. 构建产品素材管理体系，支持产品信息的录入与维护
2. 实现项目级产品素材引用，与文档素材协同使用
3. 在正文生成时自动注入产品信息，生成符合实际的技术响应内容

### 3.4 名词解释

| 术语 | 说明 |
|------|------|
| 产品素材 | 用户录入的产品信息，包含名称、型号、参数、图片等结构化数据 |
| 文档素材 | 现有的企业素材库文档，如历史标书、企业介绍等 |
| 货物类项目 | 以采购设备、材料、物资为主的招标项目，区别于工程施工类 |
| 产品彩页 | 产品的宣传图片或产品手册图片 |

## 7. 全局说明

### 7.1 交互规则

- 所有弹窗均采用模态框形式，背景遮罩

## 8. 功能详细描述

### 8.1 产品素材管理入口

**功能说明**

在现有素材库管理模块中新增"产品素材"入口，用户可进入产品素材管理页面进行产品信息的管理。

**前置条件**

- 用户已登录
- 用户具有企业/团队权限（产品素材归属企业/团队，非个人）

**页面结构**

```
素材库管理
├── 文档素材（现有）
└── 产品素材（新增）
```

**交互规则**

- 操作：点击"产品素材"菜单
- 反馈：跳转至产品素材列表页面

```
│                                                         │
```

**字段说明**

| 字段名称 | 字段类型 | 必填 | 录入方式 | 限制规则 |
|----------|----------|------|----------|----------|

| 技术参数 | 文本 | 否 | 多行文本输入 | 最大2000字符，支持换行 |
| 详细介绍 | 文本 | 否 | 多行文本输入 | 最大5000字符，支持换行 |
| 认证资质 | 文本 | 否 | 多行文本输入 | 最大1000字符，支持换行 |
| 产品图片 | 图片 | 否 | 图库选择器 | 最多5张 |
| 产品彩页 | 图片 | 否 | 图库选择器 | 最多5张 |


**交互规则**

| 操作 | 触发条件 | 交互反馈 |
|------|----------|----------|
| 打开弹窗 | 点击「新增产品」 | 弹出模态框，宽度600-800px，高度自适应 |
| 选择产品类型 | 点击下拉框 | 展示历史类型列表，支持输入筛选，支持手动输入新类型 |

| 移除已选图片 | 点击缩略图右上角× | 直接移除，不弹确认 |

| 异常场景 | 处理方式 |

| 图片选择失败 | 提示"图片加载失败，请重试" |
| 保存失败 | 提示具体失败原因，保留填写数据 |

**后置条件**

- 产品信息保存成功
- 产品图片/彩页同步存入私有图库
- 产品列表刷新显示新数据

### 8.3 产品录入 - AI智能提取

**功能说明**

用户上传产品资料文件（PDF/Word/Excel），AI自动识别并提取产品信息，批量创建产品素材。

**前置条件**

- 用户已进入产品素材管理页面
- 用户具有编辑权限

**支持格式**

| 格式 | 扩展名 | 说明 |
|------|--------|------|
| PDF | .pdf | 产品手册、彩页等 |
| Word | .doc, .docx | 产品说明书等 |
| Excel | .xls, .xlsx | 产品参数表、报价单等 |

**业务流程**

```
开始
  │
  ▼
点击「AI导入产品」按钮
  │
  ▼
选择/拖拽上传文件（支持批量）
  │
  ▼
系统处理：
  1. 文件内容转文本
  2. AI分析提取产品信息
  │
  ├── 提取成功 ──→ 产品入库 → 列表展示 → 提示"成功导入X款产品"
  │
  └── 提取失败 ──→ 提示"未能识别有效产品信息，建议手动录入"
```

**交互规则**

| 操作 | 触发条件 | 交互反馈 |
|------|----------|----------|
| 上传文件 | 拖拽或点击选择 | 显示上传进度条 |
| AI处理中 | 文件上传完成 | 显示"AI正在分析中..."加载状态 |
| 处理完成 | AI提取成功 | Toast提示导入数量，列表刷新 |
| 处理失败 | AI无法识别 | 弹窗提示失败原因和建议 |

**异常处理**

| 异常场景 | 处理方式 |
|----------|----------|
| 文件格式不支持 | 提示"暂不支持该文件格式，请上传PDF/Word/Excel文件" |
| 文件过大 | 提示"文件大小不能超过20MB" |
| AI提取无结果 | 提示"未能识别有效产品信息，建议手动录入" |
| 部分提取失败 | 提示"成功导入X款产品，Y份文件未能识别" |

**后置条件**

- 提取成功的产品信息入库
- 用户可在列表中查看和编辑

### 8.6 与私有图库的打通

### 8.7 Prompt注入策略

**功能说明**

在正文生成时，将用户选择的产品素材信息注入到AI的Prompt中，指导AI基于真实产品参数编写内容。

**注入位置**

产品素材信息与文档素材信息统一放在Prompt末尾的【参考资料】区域。

**注入格式**

```
【参考资料】

一、企业素材库文档
[文档素材内容]

二、产品素材信息
本项目涉及以下产品：

【办公家具】
- 产品名称：智能办公椅
  型号规格：ZY-001
  单位：把
  品牌：智优科技
  技术参数：座高450-550mm可调，座宽480mm，座深450mm，承重150kg...
  详细介绍：采用人体工学设计，支持腰部调节，透气网布材质...
  认证资质：通过GB/T 3325-2017检验

【电子设备】
- 产品名称：激光打印机
  型号规格：HP-1020
  单位：台
  品牌：惠普
  技术参数：打印速度14页/分钟，分辨率600×600dpi...
  详细介绍：紧凑型设计，适合中小型办公室使用...
  认证资质：3C认证、能效标识

请根据上述产品信息编写正文，涉及具体产品时请准确引用参数。
```

**注入策略**

| 阶段 | 策略 | 说明 |
|------|------|------|
| MVP阶段 | 全量注入 | 将所有已选产品信息注入Prompt |
| 后续优化 | 按需注入 | 根据章节内容智能匹配相关产品 |

## 9. 数据需求

### 9.1 埋点目的

- 了解产品素材功能的使用情况和使用频率
- 分析用户行为路径，优化功能体验
- 评估功能价值，为后续迭代提供数据支撑

### 9.2 埋点事件

| 事件名称 | 触发时机 | 事件参数 |
|----------|----------|----------|
| product_add_manual | 手动新增产品成功 | product_type, has_image, has_params |
| product_add_ai | AI导入产品成功 | file_count, product_count, file_types |
| product_edit | 编辑产品成功 | product_id |
| product_delete | 删除产品成功 | product_id, delete_type(single/batch) |
| product_search | 搜索产品 | keyword |
| product_select_enter | 打开产品挑选弹窗 | project_id |
| product_select_confirm | 确认选择产品 | project_id, product_count, doc_count |
| product_view_detail | 查看产品详情 | product_id |

### 9.3 核心指标

| 指标名称 | 计算方式 | 说明 |
|----------|----------|------|
| 产品素材总量 | SUM(产品数量) | 企业/团队的产品素材库规模 |
| 产品录入成功率 | 成功录入数/尝试录入数 | 评估录入流程顺畅度 |
| AI提取成功率 | AI成功提取数/上传文件数 | 评估AI提取能力 |
| 产品引用率 | 引用产品的项目数/总项目数 | 评估功能渗透率 |
| 单项目平均引用产品数 | SUM(项目引用产品数)/项目数 | 评估使用深度 |
