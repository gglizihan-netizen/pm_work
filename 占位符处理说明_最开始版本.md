# 招标文件占位符分析与处理说明（最开始版本）

## 一、占位符类型通用分类

### 1. 括号型占位符
使用中文括号 `（）` 或英文括号 `()` 包含的占位符，是最常见的类型：
- `（招标人名称）`、`(招标人名称)`
- `（招标项目名称）`、`(项目名称)`
- `（投标人名称）`、`(投标人名称)`
- `（制造商名称）`、`(制造商名称)`
- `（姓名）`、`(姓名)`
- `（盖单位章）`、`(签字或盖章)`

**识别特征**：文本中包含 `（` 和 `）`，中间是需要替换的字段名。

### 2. 下划线型占位符
使用下划线 `_` 或空白表示需要填写的地方：
- `_年 _月 _日`
- `____年____月____日`
- 空白填空 `    `

**识别特征**：连续的 `_` 字符或明显的空白区域。

### 3. 标签+填空型
以标签开头，后面是需要填写的内容：
- `投 标 人：__________`
- `单位性质：__________`
- `地    址：__________`
- `成立时间：____年____月____日`

**识别特征**：标签+分隔符（冒号或空格）+ 下划线/空白。

### 4. 表格型占位符
表格中的填空，分为两种子类型：

#### 类型A：标签+下划线型（一行多字段）
```
| 姓 | 名：_____性 | 别：_____ |
| 年 | 龄：_____职 | 务：_____ |
```

#### 类型B：标签列+填空列（一列标签，一列填空）
```
| 投标人名称   | ________________ |
| 注册资金     | ________________ |
| 成立时间     | ________________ |
```

## 二、通用技术分析

### Word文档Run结构
Word文档中，段落文本被分割成多个 `Run` 对象，这是所有Word文档的共同特点：

```
段落文本: '投 标 人：\t（盖单位章）'
  Run0: '投'
  Run1: ' '
  Run2: '标'
  Run3: ' '
  Run4: '人'
  Run5: '：'
  Run6: '\t'
  Run7: '（盖单位章'
  Run8: '）'
```

**影响**：简单的文本替换可能无法匹配跨 Run 的内容。

### 占位符被分割的情况
占位符可能被分割到多个 Run 中：
```
Run3: '（招标人名称'
Run4: '）'
```

或
```
Run2: '投标'
Run3: '人名称'
```

## 三、通用解决方案

### 通用工具函数

```python
from docx import Document

def get_paragraph_text(para):
    """获取段落完整文本（合并所有Run）"""
    return ''.join([run.text for run in para.runs])

def set_paragraph_text(para, new_text):
    """设置段落文本（写回第一个Run，清空其他Run）"""
    if para.runs:
        para.runs[0].text = new_text
        for run in para.runs[1:]:
            run.text = ''
    else:
        para.add_run(new_text)

def set_cell_text(cell, text):
    """设置单元格文本"""
    if cell.paragraphs:
        first_para = cell.paragraphs[0]
        if first_para.runs:
            first_para.runs[0].text = text
            for run in first_para.runs[1:]:
                run.text = ''
        else:
            first_para.add_run(text)

def has_underline(run):
    """检查Run是否有下划线"""
    return run.underline or (run._element.rPr is not None and 
                              run._element.rPr.u is not None)
```

### 方案1：括号型占位符处理（通用）

```python
def replace_placeholder_in_doc(doc, placeholder_dict):
    """
    通用函数：替换文档中所有的括号型占位符
    
    Args:
        doc: Document对象
        placeholder_dict: 占位符-内容字典，如 {'（招标人名称）': '某某公司'}
    """
    # 处理段落
    for para in doc.paragraphs:
        full_text = get_paragraph_text(para)
        new_text = full_text
        replaced = False
        
        for placeholder, content in placeholder_dict.items():
            if placeholder in new_text:
                new_text = new_text.replace(placeholder, content)
                replaced = True
        
        if replaced:
            set_paragraph_text(para, new_text)
    
    # 处理表格
    for table in doc.tables:
        for row in table.rows:
            for cell in row.cells:
                for para in cell.paragraphs:
                    full_text = get_paragraph_text(para)
                    new_text = full_text
                    replaced = False
                    
                    for placeholder, content in placeholder_dict.items():
                        if placeholder in new_text:
                            new_text = new_text.replace(placeholder, content)
                            replaced = True
                    
                    if replaced:
                        set_paragraph_text(para, new_text)
```

**使用方法**：
```python
doc = Document('招标文件.docx')
placeholders = {
    '（招标人名称）': '华美建设集团有限公司',
    '（投标人名称）': '深圳市华升电梯设备有限公司',
    '（项目名称）': '某某大厦电梯采购',
}
replace_placeholder_in_doc(doc, placeholders)
doc.save('填充后.docx')
```

### 方案2：标签+填空型处理（通用）

```python
def fill_by_label(para, label, content, separator='：'):
    """
    通用函数：根据标签填充内容
    
    Args:
        para: 段落对象
        label: 标签文本，如 '投标人'
        content: 要填充的内容
        separator: 分隔符，通常是 '：' 或 ':'
    """
    full_text = get_paragraph_text(para)
    
    # 支持多种标签变体
    label_variants = [
        label + separator,
        label.replace(' ', '') + separator,  # 去除空格
        label.replace('  ', ' ') + separator,  # 多空格变单空格
    ]
    
    for label_variant in label_variants:
        if label_variant in full_text and content not in full_text:
            # 在标签后填充内容
            new_text = full_text.replace(label_variant, label_variant + content, 1)
            set_paragraph_text(para, new_text)
            return True
    
    return False

# 使用示例
for para in doc.paragraphs:
    fill_by_label(para, '投 标 人', '深圳市华升电梯设备有限公司')
    fill_by_label(para, '单位性质', '有限责任公司')
    fill_by_label(para, '地    址', '深圳市南山区...')
```

### 方案3：表格通用处理

#### 方法A：通过标签动态定位（推荐，最通用）

```python
def fill_table_by_label(table, label_dict):
    """
    通用函数：通过标签查找并填充表格
    
    Args:
        table: 表格对象
        label_dict: {标签: 内容} 字典
    """
    for row in table.rows:
        # 假设第一列是标签列
        if len(row.cells) >= 2:
            label_cell = row.cells[0]
            label_text = get_paragraph_text(label_cell.paragraphs[0]).strip()
            
            # 遍历所有可能的标签变体
            for label, content in label_dict.items():
                # 支持标签的部分匹配（如 "投标人名称" 匹配 "投 标 人 名 称"）
                label_normalized = label.replace(' ', '').replace('\t', '')
                cell_normalized = label_text.replace(' ', '').replace('\t', '')
                
                if label_normalized in cell_normalized or cell_normalized in label_normalized:
                    # 找到匹配，填充到第二列
                    value_cell = row.cells[1]
                    set_cell_text(value_cell, content)
                    break

def fill_all_tables(doc, label_dict):
    """遍历文档所有表格，填充匹配的标签"""
    for table in doc.tables:
        fill_table_by_label(table, label_dict)

# 使用示例
label_dict = {
    '投标人名称': '深圳市华升电梯设备有限公司',
    '法定代表人': '张伟',
    '注册资金': '人民币捌仟万元整',
    '成立时间': '2010年08月18日',
}
fill_all_tables(doc, label_dict)
```

#### 方法B：通过行/列索引定位（适用于已知结构的表格）

```python
def fill_table_by_index(table, fill_map):
    """
    通过行列索引填充表格
    
    Args:
        table: 表格对象
        fill_map: {(行, 列): 内容} 字典
    """
    for (row_idx, col_idx), content in fill_map.items():
        if row_idx < len(table.rows):
            row = table.rows[row_idx]
            if col_idx < len(row.cells):
                set_cell_text(row.cells[col_idx], content)

# 使用示例（需要先分析表格结构）
table = doc.tables[0]  # 第一个表格
fill_map = {
    (0, 1): '张伟',   # 第1行第2列：姓名
    (0, 2): '男',     # 第1行第3列：性别
    (1, 1): '48',     # 第2行第2列：年龄
    (1, 2): '董事长', # 第2行第3列：职务
}
fill_table_by_index(table, fill_map)
```

#### 方法C：智能混合定位（最灵活）

```python
def smart_fill_table(table, data_dict):
    """
    智能填充表格：优先通过标签匹配，否则通过常见模式识别
    """
    for row_idx, row in enumerate(table.rows):
        if len(row.cells) == 0:
            continue
            
        # 获取第一列文本（通常是标签）
        first_cell_text = get_paragraph_text(row.cells[0].paragraphs[0])
        first_cell_normalized = first_cell_text.replace(' ', '').replace('\t', '').replace('：', '').replace(':', '')
        
        # 尝试匹配数据字典
        for key, value in data_dict.items():
            key_normalized = key.replace(' ', '').replace('\t', '')
            
            if key_normalized in first_cell_normalized:
                # 标签匹配成功，填充到第二列或后续列
                if len(row.cells) >= 2:
                    # 找到第一个为空的单元格填充
                    for cell in row.cells[1:]:
                        cell_text = get_paragraph_text(cell.paragraphs[0])
                        if not cell_text.strip() or '____' in cell_text or '____' in cell_text:
                            set_cell_text(cell, value)
                            break
                break

# 使用示例
data = {
    '投标人名称': '深圳市华升电梯设备有限公司',
    '单位性质': '有限责任公司',
    '地址': '深圳市南山区...',
    '注册资金': '8000万元',
    '法定代表人': '张伟',
}
for table in doc.tables:
    smart_fill_table(table, data)
```

## 四、通用处理流程

### 标准处理流程（适用于任何招标文件）

```python
def process_bid_document(input_path, output_path, data):
    """
    通用招标文件处理流程
    
    Args:
        input_path: 输入文件路径
        output_path: 输出文件路径
        data: 填充数据字典
    """
    doc = Document(input_path)
    
    # 步骤1：替换所有括号型占位符
    placeholder_dict = {f'（{k}）': v for k, v in data.items()}
    placeholder_dict.update({f'({k})': v for k, v in data.items()})
    replace_placeholder_in_doc(doc, placeholder_dict)
    
    # 步骤2：处理标签+填空型段落
    label_mapping = {
        '投 标 人': data.get('投标人名称'),
        '单位性质': data.get('单位性质'),
        '地    址': data.get('地址'),
        '经营期限': data.get('经营期限'),
    }
    for para in doc.paragraphs:
        for label, content in label_mapping.items():
            if content:
                fill_by_label(para, label, content)
    
    # 步骤3：处理所有表格
    for table in doc.tables:
        smart_fill_table(table, data)
    
    doc.save(output_path)

# 使用
data = {
    '投标人名称': '您的公司',
    '法定代表人': '您的姓名',
    '地址': '您的地址',
    # ... 其他字段
}
process_bid_document('模板.docx', '输出.docx', data)
```

## 五、常见变体及处理

### 变体1：标签格式不同
```
原始: 投 标 人：
变体: 投标人：
变体: 投 标 人:
处理: 使用归一化比较（去除空格和分隔符）
```

### 变体2：表格结构不同
```
类型A: 一行两列（标签|值）
类型B: 一行四列（标签|值|标签|值）
类型C: 标签在第一列，值跨多列
处理: 智能检测单元格数量和内容分布
```

### 变体3：下划线格式不同
```
格式A: _年 _月 _日
格式B: ____年____月____日
格式C: 空白（无下划线）
处理: 统一使用正则表达式匹配日期格式
```

## 六、处理流程

1. **全局占位符替换**：遍历所有段落和表格，替换括号型占位符
2. **投标函特殊处理**：
   - 招标人、投标人、法定代表人
   - 地址、邮编、电话、传真
   - 日期（下划线替换）
3. **法定代表人身份证明**：
   - 投标人、单位性质、地址
   - 成立时间、经营期限
   - 法人信息（表格1）
4. **授权委托书**：
   - 第一个（姓名）→ 法定代表人
   - 第二个（姓名）→ 被委托人
   - 委托期限
5. **制造商授权书**：
   - 制造商信息
   - 授权期限
6. **资格审查资料表格**：
   - **表格1：法定代表人信息表**（表格型标签+下划线）
     ```python
     table = doc.tables[0]
     # 第1行：姓名、性别
     set_cell_text(table.rows[0].cells[1], '名：张伟\t性')
     set_cell_text(table.rows[0].cells[2], '别：男\t')
     # 第2行：年龄、职务
     set_cell_text(table.rows[1].cells[1], '龄：48\t职')
     set_cell_text(table.rows[1].cells[2], '务：董事长、总经理\t')
     ```
   
   - **表格4：投标人基本情况表**（纯文本填空型）
     ```python
     table = doc.tables[3]
     set_cell_text(table.rows[0].cells[1], data['投标人名称'])
     set_cell_text(table.rows[1].cells[1], data['注册资金'])
     set_cell_text(table.rows[1].cells[4], data['成立时间'])
     set_cell_text(table.rows[2].cells[1], data['地址'])
     set_cell_text(table.rows[3].cells[1], data['邮政编码'])
     set_cell_text(table.rows[3].cells[4], data['员工总数'])
     # ... 其他字段
     ```
   
   - **表格5：近年类似项目情况表**（纯文本填空型）
     ```python
     table = doc.tables[4]
     set_cell_text(table.rows[0].cells[1], data['设备名称'])
     set_cell_text(table.rows[1].cells[1], '乘客电梯 HLK-1000/1.75')
     # ... 其他字段
     ```
   
   - **表格6：项目负责人简历表**（纯文本填空型）
     ```python
     table = doc.tables[5]
     set_cell_text(table.rows[0].cells[1], data['项目负责人姓名'])
     set_cell_text(table.rows[0].cells[3], data['项目负责人年龄'])
     set_cell_text(table.rows[1].cells[1], data['项目负责人职称'])
     # ... 其他字段
     ```
